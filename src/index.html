Eyes up, Guardian!

Introduction



Destiny Objective Tracker, or DOT for short, is an application designed to help Guardians keep track of various objectives from within Destiny 2.


Currently, DOT is available in alpha preview for Android devices. If you'd like to take part in the alpha, click here or use the "Alpha" link in the navigation menu.

Features

DOT is able to keep track of the following for you:

  • Bounties
  • Quests
  • Armor Based Objectives (Ex: Solstice of Heroes armor)
Here's a peek at some upcoming planned features!
  • Challenges & Milestones
  • Seasonal Challenges
  • Catalysts
  • Vendors (preview what quests/bounties vendors are selling)
  • Triumphs
